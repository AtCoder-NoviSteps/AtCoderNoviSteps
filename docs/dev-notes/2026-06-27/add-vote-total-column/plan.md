# 投票総数列の追加（グレード管理ページ）

## 概要

グレード管理ページ（`/tasks/grade`）の一覧テーブルに「投票総数」列を追加する。
「グレード（ユーザ投票）」列の右側に配置し、0票の場合は `-` を表示する。

## 設計

- `TaskWithVoteInfo.voteTotal: number` はサービス層で既に計算済み
- `filteredTasks` 経由で各行からアクセス可能
- 変更は `+page.svelte` のみで完結（型・サービス変更なし）

## 却下した選択肢

- `42票` のように単位を付ける → 数値のみの方がテーブルでコンパクト

## タスク

- [ ] テーブルヘッダーに「投票総数」列を追加（「グレード（ユーザ投票）」の右）
- [ ] 各行に `task.voteTotal === 0 ? '-' : task.voteTotal` を表示するセルを追加
- [ ] 空状態の `colspan` を 4 → 5 に更新（2箇所）

## 変更ファイル

`src/routes/(admin)/tasks/grade/+page.svelte`

## 検証

```bash
pnpm check
```

ブラウザで `/tasks/grade` を開き、検索後に投票総数列が表示されることを確認。
